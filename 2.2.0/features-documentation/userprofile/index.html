Overview

Creating and updating profile on Sunbird platform is a seamless process. This allow users to extend and provide informations related to their experience, skills, education, awards etc. Providing this information helps the system provide personalized recommendations for courses to upgrade your skills, selects you for custom programs, connects you to others in the community, build bridges for collaboration, etc. You may choose to create, update or edit your profile information at any time.

Note: Enter details in all mandatory fields. You cannot submit the form in case you miss entering details in any of the mandatory fields. In such a case, an error message is displayed. Close the message box and provide the required details, before you proceed to submit the form.

Prerequisites

Step Screen
1. You are logged in
2. You are currently on Home page
3. You want to add or edit your personal information , you have clicked on Profile tab

Viewing Profile and Adding Image

Step Screen
1. Login with the registered credential
2. Click Profile tab on the header
Or
3. Click Profile to the right
1. Depending on your organization, you can see few information are already available, viz: Username, City you belong to etc
2. Click the Edit icon on your profile, to add profile image. For details on adding image to profile, refer Adding Images
3. You can also see your Last Login Time

Note: Maximum image upload size is 4 MB

Selecting Organization Types

Step Screen
1. Click Search dropdown button
2. Select Organizations from the dropdown list
3. Click Filter icon
4. Select Organization Type from the dropdown list
5. Click Apply to associate an organization with the organization type
6. Click Reset button to select attributes again

Filtering Profile by Attributes

Step Screen
1. Click Search dropdown button
2. Select Users from the dropdown list
3. Click Filter icon
4. You can filter the profile by:
  • Grades
  • Medium
  • Subjects
  • Roles
  • Location

    • 5. Click Apply to tag an organization with the organization type
    6. Click Reset to select attributes again

    Step Screen

    Add or edit your experience details, such as work title, organization, designation, joining date, subject taught etc. You can show or hide your experience details from others

    1. Click Edit

    1. Click View More dropdown in Skill Tags section
    2. Click + icon to endorse for a particular skill

    1. Add or edit personal information such as name, phone number, date of birth, location, languages known, etc
